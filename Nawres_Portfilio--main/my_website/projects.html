Image Recoloring with conditional GANs

This project aims to explore innovative approaches to image recoloration using conditional Generative Adversarial Networks (cGANs). The objectives include: Developing three unique models based on the Pix2Pix framework to recolor grayscale images into vibrant RGB or LAB formats. Introducing enhancements such as perceptual loss (LPIPS) and WGAN loss to improve the quality and diversity of generated images. Leveraging a U-Net generator and PatchGAN discriminator for accurate pixel-level mappings and robust adversarial training. Implementing a robust evaluation framework utilizing metrics such as FID, IS, SSIM, PSNR, and MAE to assess performance and diversity. The project highlights the challenges of recoloring images, including ensuring high fidelity, preserving structural consistency, and maintaining perceptual quality. Extensive experiments demonstrate that the model with perceptual loss yields the most diverse and visually appealing recolored images.

Image Completion with Feature Descriptors

This project focuses on developing a computer vision algorithm for **image completion**, which automatically finds and corrects missing or corrupted areas in an image using provided patches. The project involves: Loading corrupted images and patches and preprocessing them with equalization, Gaussian blur, and sharpening techniques. Extracting SIFT features from images and patches to identify keypoints and descriptors. Matching features between the image and patches using ORB and SIFT descriptors and refining matches with distance ratio tests. Computing the affine transformation matrix with RANSAC to overlay patches onto the corrupted image. Using the homography matrix to seamlessly blend patches into the image, restoring the corrupted regions. The final implementation provides accurate restoration of missing image regions, leveraging robust feature matching and transformation techniques.
